# 02 — 指令、函数、配置项速查表

## 模板指令

| 语法 | 说明 |
|------|------|
| `${expr}` | 表达式插值，输出求值结果 |
| `<#if cond>...</#if>` | 条件分支，支持 `<#else>` 和 `<#elseif>` |
| `<#for x:items>...</#for>` | 遍历 `items`，元素绑定为 `x`；`<#list>` 为其别名；支持 `<#else>`（空列表时执行） |
| `<#continue[depth:][cond]>` | 跳过当前迭代。`<#continue 2>` 跳过两层，`<#continue cond>` 条件成立时跳过 |
| `<#break[depth:][cond]>` | 退出循环。`<#break 2>` 退出两层，`<#break cond>` 条件成立时退出 |
| `<#macro name params>...</#macro>` | 宏定义，参数按空白分割，不支持默认值和引号分组 |
| `<@name arg1 arg2/>` | 宏调用，参数以空白分隔 |
| `<#include "path">` | 引入子模板，默认相对于当前文件所在目录；以 `/` 开头则视为操作系统绝对路径 |
| `<#meta>...</#meta>` | 元数据块 |
| `<#-- comment -->` | 注释，不输出 |

## Meta 标签

以下标签写在 `<#meta>...</#meta>` 内部：

| 标签 | 说明 |
|------|------|
| `@Param{ key: value, ... }` | 模板参数定义 |
| `@Cartesian{ name: list, ... }` | 笛卡尔积轴定义（轴名 → 值列表或函数调用） |
| `@Path{ expr }` | 输出路径表达式（支持函数调用和引用） |
| `@SkipWhen{ expr }` | 跳过条件，布尔表达式为 true 时跳过本次生成 |
| `@Config{ key: value, ... }` | 模板级行为配置 |

## 内置函数

### 比较逻辑

| 函数 | 中缀写法 | 说明 |
|------|---------|------|
| `greaterThan(a, b)` | `a greaterThan b` | `a > b` |
| `lessThan(a, b)` | `a lessThan b` | `a < b` |
| `greaterThanOrEquals(a, b)` | `a >= b` | `a >= b` |
| `lessThanOrEquals(a, b)` | `a <= b` | `a <= b` |
| `equals(a, b)` | `a equals b` | `a == b`（深度比较） |
| `notEquals(a, b)` | `a notEquals b` | `a != b` |
| `and(a, b)` | `a and b` | 逻辑与 |
| `or(a, b)` | `a or b` | 逻辑或 |
| `not(a)` | `not a` | 逻辑非 |

### 字符串

| 函数 | arity | 说明 |
|------|-------|------|
| `capital(s)` | 1 | 首字母大写 |
| `lower(s)` | 1 | 转小写 |
| `upper(s)` | 1 | 转大写 |
| `javaStr(s)` | 1 | 转为 Java 字符串字面量（转义 `\`、`"`、`\n`、`\r`、`\t`） |
| `concat(a, b)` | 2 | 拼接 |
| `contains(s, sub)` | 2 | 是否包含子串 |
| `replace(s, from, to)` | 3 | 替换 |
| `startsWith(s, prefix)` | 2 | 是否以 prefix 开头 |
| `repeat(s, n)` | 2 | 重复 n 次 |
| `join(sep, items...)` | 变参 | 用分隔符拼接 |

### 判空

| 函数 | arity | 说明 |
|------|-------|------|
| `notNull(x)` | 1 | `x != null` |
| `isNull(x)` | 1 | `x == null` |
| `isEmpty(x)` | 1 | String/Collection/Map 为空 |
| `isBlank(x)` | 1 | null 或空白字符串 |

### 算术

| 函数 | arity | 说明 |
|------|-------|------|
| `plus(a, b)` | 2 | 加（若含字符串则拼接） |
| `minus(a, b)` | 2 | 减 |

### 范围序列

| 函数 | arity | 说明 |
|------|-------|------|
| `range(start, end)` | 2 | 闭区间 `[start, end]` 的数字列表 |
| `seqJoin(list, prefix, sep, suffix)` | 4 | 模板填充式拼接 |

### 工具

| 函数 | arity | 说明 |
|------|-------|------|
| `default(x, fallback)` | 2 | `x != null` 则 `x`，否则 `fallback` |
| `length(x)` | 1 | String/Collection/Map/Array 长度 |
| `now(pattern)` | 1 | 当前时间，按 pattern 格式化 |
| `packageToPath(pkg)` | 1 | `.` 替换为 `/` |

### 组合生成

| 函数 | arity | 说明 |
|------|-------|------|
| `selfCartesian(list, n)` | 2 | 从 list 中取 n 个元素的可重复排列（笛卡尔积） |
| `permutation(list, n)` | 2 | 从 list 中取 n 个元素的全排列（不重复） |
| `combination(list, n)` | 2 | 从 list 中取 n 个元素的组合 |
| `multiCombination(list, n)` | 2 | 从 list 中取 n 个元素的可重复组合 |
| `cartesian(lists...)` | 变参 | 多个列表的笛卡尔积 |
| `concatList(a, b)` | 2 | 拼接两个列表 |
| `concatField(list, field)` | 2 | 提取列表中每个对象的字段拼接成新列表 |
| `sortBy(list, field)` | 2 | 按字段排序 |
