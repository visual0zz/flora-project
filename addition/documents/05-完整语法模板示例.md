# 05 — 完整语法模板示例

以下示例用到了 Ramet 引擎的大部分语法元素。

```ramet
<#meta>
@Param{
  pkg: "com.example.gen",
  className: "DemoFactory"
}
@Cartesian{
  T: selfCartesian([Int, Long, Float], 1)
}
@Path{ concat(packageToPath(pkg), "/", className, T.name, ".java") }
@Config{ autoWarning: true }
</#meta>
package ${pkg};

import com.example.gen.core.BaseFactory;

/**
 * ${className}${T.name} — 由模板自动生成。
 */
public final class ${className}${T.name} extends BaseFactory {

    <#-- 私有构造器 -->
    private ${className}${T.name}() {}

    /**
     * 创建一个 ${T.type} 类型的实例。
     */
    public static ${className}${T.name} of(${T.type} value) {
        return new ${className}${T.name}(value);
    }

    <#macro fieldDef type name>
    private final ${type} ${name};
    </#macro>

    <@fieldDef T.type "value"/>

    public ${T.type} getValue() {
        return value;
    }

    <#if T.name equals "Float">
    public boolean isNaN() {
        return java.lang.Float.isNaN(value);
    }
    </#if>

    <#list i:range(1, 3)>
    public ${T.type} multiplyBy${i}() {
        <#if T.name equals "Float">
        return value * ${i}.0f;
        <#else>
        return value * ${i};
        </#if>
    }
    </#list>

    @Override
    public String toString() {
        return "<#" + "${T.name}" + " value=" + value + ">";
    }
}
```

## 运行方式

```bash
# 从模板目录生成
java -jar flora-ramet-0.1.jar \
  /path/to/templates \
  /path/to/output

# 或使用 Maven 插件
mvn io.gitee.visual0zz:flora-ramet-maven-plugin:generate \
  -Dramet.templatesDir=src/main/templates \
  -Dramet.outputDir=src/main/java
```

## 此模板涉及的语法元素

| 语法 | 用途 |
|------|------|
| `<#meta>...</#meta>` | 元数据块声明 |
| `@Cartesian{ ... }` | 定义笛卡尔积轴 `T` |
| `@Path{ ... }` | 动态生成输出路径 |
| `@Config{ ... }` | 开启自动警告注释 |
| `${pkg}` `${className}` `${T.type}` | 变量插值 |
| `concat()` `packageToPath()` `selfCartesian()` `range()` | 内置函数调用 |
| `<#if T.name equals "Float">` | 条件分支（中缀表达式） |
| `<#else>` | 条件否则 |
| `<#list i:range(1, 3)>` | 循环 |
| `<#macro fieldDef type name>` | 宏定义 |
| `<@fieldDef T.type "value"/>` | 宏调用 |
| `<#-- 注释 -->` | 注释 |
| `"${T.name}"` | 字符串字面量（Java 转义，输出 `${xxx}`） |
